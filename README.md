# Job Tracker

This is my Job Tracker project. I made this to keep track of job applications in one place.

## What it does

- User can register and login
- Add job details
- View all applied jobs on dashboard
- Helps in organizing job search

## Tech used

- React (frontend)
- Node.js + Express (backend)
- MongoDB (database)
